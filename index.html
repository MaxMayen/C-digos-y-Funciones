ETIQUETAS HTML (HTML5)




Estructura Básica

•html: Raíz de un documento HTML

•head: Contiene metadatos, enlaces a scripts, estilos, etc.

•title: Título del documento

•body: Cuerpo del documento



Metadatos

•meta: Metadatos como codificación, descripción, autor, etc.

•link: Enlaza archivos externos como hojas de estilo (CSS)

•style: Define estilos CSS internos

•script: Código JavaScript

Texto y Contenido

•h1 a h6: Encabezados (de mayor a menor)

•p: Párrafo

•br: Salto de línea

•hr: Línea horizontal

•strong: Texto en negrita semántica

•b: Texto en negrita

•em: Texto en cursiva semántica

•i: Texto en cursiva

•u: Texto subrayado

•small: Texto más pequeño

•mark: Texto resaltado

•blockquote: Cita textual

•code: Texto con formato de código

•pre: Texto preformateado

•abbr: Abreviatura (con tooltip)

•sub / sup: Subíndice / Superíndice

Hipervínculos

•a: Enlace a otra página o parte del documento

Imágenes y Multimedia

•img: Inserta imágenes

•audio: Inserta audio

•video: Insertar video

•source: Fuente alternativa para audio o video

•track: Subtítulos para video

Contenedores

•div: Contenedor genérico de bloque

•span: Contenedor genérico en línea

•section: Sección de contenido

•article: Artículo independiente

•header: Encabezado de una sección

•footer: Pie de página

•nav: Menú de navegación

•aside: Contenido complementario

Listas

•ul: Lista desordenada

•ol: Lista ordenada

•li: Elemento de lista

•dl: Lista de definiciones

•dt/dd: Término/Definición

Tablas

•table: Tabla

•thead/tbody/tfoot: Cabeza/Cuerpo/Pie de tabla

•tr: Fila

•th: Encabezado de celda

•td: Celda de datos

•caption: Título de tabla

•col/colgroup: Columnas de tabla

Formularios

•form: Formulario

•input: Campo de entrada

•label: Etiqueta para un campo

•textaera: Área de texto

•button: Botón

•select: Menú desplegable

•option: Opción dentro de "select"

•fidelset: Agrupa elementos del formulario

•legend: Título para "fidelset"

•datalist: Lista de opciones sugeridas

•output: Muestra resultados de cálculos

Elementos Interactivos

•details: Crea un panel desplegable

•summary: Título visible del "details"

•dialog: Cuadro de diálogo (modal)

•menu/menuitem: Menú Contextual

Contenido Semántico

•main: Contenido Principal

•figure: Agrupa contenido visual con leyenda

•figcaption: Leyenda de "figure"

•time: Fecha/hora

•address: Dirección física o de contacto

Scripts y Enlaces

•script: Código JavaScript

•noscript: Contenido alternativo si JS está deshabilitado

Otros

•!DOCTYPE html: Declaración del tipo de documento

•!--comentario--: Comentarios en el codigo
